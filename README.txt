Thrift Store Finder App

This app helps users find thrift stores based on their location. It uses the Google Maps API to list all the thrift stores in the United States. The app includes features like search, filter, store details, user reviews, and favorites.

To compile and test the app, follow these steps:

1. Download and extract the thrift_store_finder.zip file.
2. Install Flutter and Dart on your machine.
3. Open a terminal window and navigate to the project directory.
4. Run 'flutter pub get' to install the dependencies.
5. Run 'flutter run' to run the app.

Please note that you will need a device or an emulator to run the app.
